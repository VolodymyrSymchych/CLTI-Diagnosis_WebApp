# Короткий огляд проєкту CLTI Diagnosis

## 🎯 Мета проєкту
Веб-застосунок для медичних фахівців (ангіохирургів, сосудистих хірургів) для діагностики та оцінки **Critical Limb Threatening Ischemia (CLTI)**.

## 📊 Поточний стан

### ✅ Реалізовано
- ✅ Аутентифікація та авторизація (JWT + Cookies)
- ✅ WiFI класифікація (Wound, Ischemia, foot Infection)
- ✅ CRAB оцінка (Periprocedural Mortality)
- ✅ 2YLE шкала (Two-Year Life Expectancy)
- ✅ GLASS класифікація (Global Anatomic Staging System)
- ✅ Гемодинамічні показники (КПІ/ППІ)
- ✅ Оцінка реваскуляризації
- ✅ AI асистент (OpenAI)
- ✅ Управління справами пацієнтів
- ✅ Завантаження фото

### ⚠️ Потрібно зробити
- ⚠️ Тестування (unit, integration, E2E)
- ⚠️ Рефакторинг архітектури (Clean Architecture)
- ⚠️ Документація API (Swagger)
- ⚠️ Оптимізація продуктивності
- ⚠️ Покращення безпеки

## 🏗️ Архітектура

**Backend**: ASP.NET Core 9.0 (Blazor Server)  
**Frontend**: Blazor WebAssembly 9.0  
**Database**: SQL Server  
**Authentication**: JWT + Cookie Authentication
