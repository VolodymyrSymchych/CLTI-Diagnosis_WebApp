## Огляд проєкту

### Мета проєкту
**CLTI Diagnosis** - веб-застосунок для медичних фахівців (ангіохирургів, сосудистих хірургів) для діагностики та оцінки **Critical Limb Threatening Ischemia (CLTI)** - критичної загрози ішемії нижніх кінцівок.

### Технологічний стек
- **Backend**: ASP.NET Core 9.0 (Blazor Server)
- **Frontend**: Blazor WebAssembly 9.0
- **База даних**: SQL Server
- **Аутентифікація**: JWT + Cookie Authentication
- **Логування**: Serilog
- **AI**: OpenAI API Integration

### Цільова аудиторія
- Ангіохирурги та сосудисті хірурги
- Лікарі, які працюють з пацієнтами зі складними ураженнями нижніх кінцівок
- Медичні заклади та клініки
- Дослідники в галузі кардіології та хірургії судин

#### 2. Діагностичні модулі
- ✅ **WiFI Класифікація** (Wound, Ischemia, foot Infection)
  - Оцінка рани (W)
  - Оцінка ішемії (I)
  - Оцінка інфекції (fI)
  - Автоматичний розрахунок клінічної стадії (0-4)
  
- ✅ **CRAB Оцінка** (Periprocedural Mortality)
  - Розрахунок ризику хірургічного втручання
  - Врахування віку, анамнезу, функціонального стану
  
- ✅ **2YLE Шкала** (Two-Year Life Expectancy)
  - Прогнози дворічної виживаності пацієнта
  
- ✅ **GLASS Класифікація** (Global Anatomic Staging System)
  - Визначення стадії (I, II) та підстадій (A, B)
  - Оцінка аорто-клубового сегмента
  - Оцінка стегново-підколінкового сегмента
  - Оцінка інфрапоплітеального сегмента
  - Фінальна стадія GLASS
  - Підкісточкова (стопна) хвороба

- ✅ **Гемодинамічні показники**
  - КПІ (Клінічний Показник Ішемії) / ABI
  - ППІ (Показник Перфузії Ікр) / PPI

- ✅ **Оцінка реваскуляризації**
  - Вибір методу реваскуляризації
  - Оцінка підходу (ендоваскулярний vs хірургічний)

## Структура проєкту

### Backend (CLTI.Diagnosis)
```
CLTI.Diagnosis/
├── Components/          # Blazor Server компоненти
│   ├── Account/        # Аутентифікація
│   ├── Layout/         # Макети сторінок
│   └── Pages/          # Сторінки
├── Core/
│   ├── Application/    # Бізнес-логіка (порожня структура)
│   └── Domain/         # Доменні сутності
├── Data/               # DbContext
├── Infrastructure/
│   ├── Data/           # Репозиторії (порожня структура)
│   └── Services/       # Сервіси (PasswordHasher, SessionStorage)
├── Models/
│   └── DTOs/           # Data Transfer Objects
├── Web/
│   ├── Controllers/    # API контролери
│   └── Middleware/     # Middleware компоненти
└── Migrations/         # EF Core міграції
```

### Frontend (CLTI.Diagnosis.Client)
```
CLTI.Diagnosis.Client/
├── Account/            # Сторінки аутентифікації
├── Algoritm/           # Алгоритми діагностики
├── Components/         # Переіспользувані компоненти UI
├── Features/
│   └── Diagnosis/      # Основні функції діагностики
│       ├── Models/     # Моделі даних
│       ├── Pages/      # Сторінки діагностики
│       ├── Services/   # Калькулятори та сервіси
│       └── Store/      # State management
├── Infrastructure/     # HTTP клієнти, Auth, State
└── Shared/             # Спільні компоненти
```
